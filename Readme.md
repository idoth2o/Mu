#  Multi platform Unified OS: Mu OS
	Arch:x86/x64(LP64)
	For study x64 & hobby develop my OS
    (x86/x64環境で動作する勉強＆ホビー用の自作OS)
    
## Prerequisites

- [gcc4.8.2 or later]
- [GNU GRUB 1.98]

## Support on Develop
	OSX 10.9(T.B.D)
	Linux Ubuntu 12.04 LTS(T.B.D)
	Windows(Cygwin)

## Build
	Make boot image for x86
	$ make iso
	
	Make boot image for x64
	$ make arch=x64 iso	

## Status
	build x86/x64 boot image & boot
